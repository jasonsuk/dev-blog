## File description

<a id="files"></a>

The application is based on MERN stack. React used for the client side (frontend) and MongoDB, Express and Node for the server side (backend)

There are two main folders:

-   `frontend`

    The client side codes using `React` framework along with Redux and Hook for state management.

-   `backend`

    The server side codes based on `Node / Express` connected to `MongoDB` database (using Mongoose Object Document Mapper)
